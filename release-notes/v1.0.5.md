# Mdora v1.0.5 (Public)

## English

This release focuses on public-readiness improvements around task-state compatibility and large-document responsiveness.

### Highlights
- Added public-facing performance controls for motion and rendering fallback in Settings/About.
- Improved editor highlight scheduling and preview pulse throttling for smoother interaction on large documents.
- Expanded task marker compatibility mapping with additional alias forms and localized-compatible labels.
- Refreshed release material for public distribution metadata and README references.

### Fixes
- Reduced unnecessary expensive UI updates under long-document load.
- Better aligned task marker handling and compatibility display across parser, editor, and preview paths.

### Installation
- Download: https://github.com/Gurara-nya/Mdora/releases/download/v1.0.5/Mdora-1.0.5-macOS.zip

### Compatibility
- macOS 14+
- Markdown tasks, links, diagrams, math blocks, tables, mermaid/plantuml/flowchart previews

- Public release date: `2026-06-07`
- Repo: https://github.com/Gurara-nya/Mdora
