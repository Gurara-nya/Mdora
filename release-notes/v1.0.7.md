# Mdora v1.0.7 (Public)

## English

This public release adds clearer visibility for long-document fallback behavior.

### Highlights
- Added visible preview degradation status indicators in the status bar.
- Refined preview configuration flow so high-load reductions are reported explicitly.

### Fixes
- Fixed inconsistent degradation feedback in cases where large-document auto-reduction is triggered.
- Kept release metadata and packaging references aligned for public distribution.

### Installation
- Download: https://github.com/Gurara-nya/Mdora/releases/download/v1.0.7/Mdora-1.0.7-macOS.zip

### Compatibility
- macOS 14+
- Markdown tasks, links, diagrams, math blocks, tables, mermaid/plantuml/flowchart previews

### Notes
- Public release date: `2026-06-07`
- Repo: https://github.com/Gurara-nya/Mdora
