# Mdora v1.0.4 (Public)

## English

Mdora 1.0.4 is a stabilization release for parser compatibility and public usability on long documents.

### Highlights
- Unified task-state semantics in `TaskState` so editor and preview share the same behavior for cycle state, muted/struck-through status, and toggle semantics.
- Expanded task marker compatibility to support hyphen and underscore in alias tokens.
- Added large-document adaptive rendering and editor fallback behavior for smoother performance under heavy markdown content.
- Added release note and changelog updates for the current public package.

### Fixes
- Removed duplicated task-state helper logic between model and UI modules.
- Tuned large-document thresholds to reduce unnecessary animations/highlight work for large files.

### Installation
- Download: https://github.com/Gurara-nya/Mdora/releases/download/v1.0.4/Mdora-1.0.4-macOS.zip

### Compatibility
- macOS 14+
- Markdown tasks, links, diagrams, math blocks, tables, mermaid/plantuml/flowchart previews

### Notes
- Public release date: `2026-06-07`
- Repo: https://github.com/Gurara-nya/Mdora

## 中文说明

Mdora 1.0.4 是一次用于兼容修复与大文档稳定性的公开稳定版发布。

### 亮点
- 将任务状态语义集中到 `TaskState`，让编辑器与预览在循环状态、禁用/划线语义、切换语义上保持一致。
- 扩展任务别名解析，支持连字符和下划线风格的 token。
- 增加大文档下的自适应降级策略，提升长文本场景下的渲染与编辑稳定性。
- 补充本次公开发布说明与更新日志内容。

### 修复
- 去除模型层与 UI 层重复的任务状态辅助逻辑，降低逻辑分叉风险。
- 调整大文档阈值，减少长文档下不必要的动画与高亮计算。
