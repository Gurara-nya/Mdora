# Mdora v1.0.2 (Public)

## English

Mdora 1.0.2 is a public readiness release focused on release metadata consistency, task state compatibility, and publication polish.

### Highlights
- Refined extended task state/model compatibility (warning, blocked, review, idea, success, done).
- Improved task marker handling consistency between parser, analyzer, and editor continuations.
- Updated public-facing metadata (README/Changelog) and release body for distribution consistency.

### Fixes
- Fixed duplicated task marker definition paths in `TaskState` and `TaskTokenKind` model helpers.
- Kept release package linkage and notes aligned with current public version.

### Installation
- Download: https://github.com/Gurara-nya/Mdora/releases/download/v1.0.2/Mdora-1.0.2-macOS.zip

### Compatibility
- macOS 14+
- Markdown tasks, links, diagrams, math blocks, tables, mermaid/plantuml/flowchart previews

### Notes
- Public release date: `2026-06-07`
- Repo: https://github.com/Gurara-nya/Mdora
