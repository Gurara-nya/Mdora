# Mdora v1.0.3 (Public)

## English

Mdora 1.0.3 is a public stabilization release focused on 2.0 preparation, parser compatibility, and rendering performance polish.

### Highlights
- Expanded task-marker compatibility for `warning`, `blocked`, `review`, `idea`, and `success` states across parser, analyzer, and editor paths.
- Added resolution-aware image preview cache keys and background loading to avoid stale/incorrect previews when size thresholds change.
- Added 2.0 preset controls for animation/table/chart/math/image thresholds in Settings.
- Updated release metadata, About/preview styling hooks, and public-facing docs for the new publish.

### Fixes
- Fixed task marker alias and constructor compatibility drift in shared model helpers.
- Reduced stale cache reuse risk for local image preview paths with different image dimension caps.

### Installation
- Download: https://github.com/Gurara-nya/Mdora/releases/download/v1.0.3/Mdora-1.0.3-macOS.zip

### Compatibility
- macOS 14+
- Markdown tasks, links, diagrams, math blocks, tables, mermaid/plantuml/flowchart previews

### Notes
- Public release date: `2026-06-07`
- Repo: https://github.com/Gurara-nya/Mdora

## 中文说明

Mdora 1.0.3 是一次公开稳定版发布，聚焦 2.0 版本的前置准备、解析兼容与渲染性能打磨。

### 亮点
- 在解析器、分析器与编辑器链路中完善 `warning`、`blocked`、`review`、`idea`、`success` 等任务状态兼容。
- 引入分辨率感知的图片预览缓存键与后台加载，避免在不同阈值下出现旧图复用问题。
- Settings 增加 2.0 预设参数，集中控制动画、表格、图谱、公式与图片阈值。
- 同步更新公开发布信息、About/预览样式映射与说明文档。

### 修复
- 修复任务标记与构造器在共享模型中的别名兼容漂移。
- 降低本地图片在不同尺寸约束下的缓存复用风险。
