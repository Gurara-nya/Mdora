# Mdora v1.0.6 (Public)

## English

This public release adds clear visibility for performance fallback behavior in large documents.

### Highlights
- Added a visible preview banner for performance degradation reasons in large-document scenarios.
- Kept release metadata and package/build versioning aligned with public distribution artifacts.

### Fixes
- Corrected the release-1.0.6 public build compatibility issue introduced during preview status UI updates.
- Improved markdown task and preview synchronization visibility on degraded rendering paths.

### Installation
- Download: https://github.com/Gurara-nya/Mdora/releases/download/v1.0.6/Mdora-1.0.6-macOS.zip

### Compatibility
- macOS 14+
- Markdown tasks, links, diagrams, math blocks, tables, mermaid/plantuml/flowchart previews

### Notes
- Public release date: `2026-06-07`
- Repo: https://github.com/Gurara-nya/Mdora
